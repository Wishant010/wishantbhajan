# Wishant Bhajan Portfolio (Next.js)

This project runs on the Next.js App Router with React, TypeScript, and Tailwind CSS. It powers the portfolio experience for wishantbhajan.nl with rich animations and 3D components.

## Getting started

```bash
npm install
npm run dev
```

## Production build

```bash
npm run build
npm run start
```

## Quality checks

```bash
npm run lint
npm run type-check
```
